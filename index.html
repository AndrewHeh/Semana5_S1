Selectores

Los selectores en CSS se utilizan para seleccionar los elementos HTML que queremos estilizar. Algunos ejemplos de selectores son:

element : Selecciona todos los elementos de ese tipo./p>

#id : Selecciona el elemento con ese ID específico.

.clase : Selecciona todos los elementos con esa clase.

Herencia

La herencia en CSS se refiere a la capacidad de algunos estilos de ser heredados por los elementos hijos desde sus elementos padres. No todas las propiedades son heredables, pero muchas de las propiedades relacionadas con el texto lo son.

Cascada

La cascada en CSS se refiere al orden en que se aplican las reglas CSS. Las reglas pueden sobrescribirse según la fuente, el origen, la importancia y el orden en el que aparecen.

Especificidad

La especificidad determina qué reglas CSS se aplican cuando hay conflictos. Es una medida de qué tan específico es un selector. Los estilos inline tienen la mayor especificidad, seguidos por los ID, clases y elementos.
